HTML/CSS → DOM API Converter

Paste HTML or a JS snippet that builds HTML via innerHTML/outerHTML. Each assignment is converted to safe createElement/appendChild calls with correct element nesting. The converter handles full JS: for/while/do loops, if/else chains, try/catch, switch/case, ternaries, .map().join(), template literals, and function inlining. Non-build code (counters, flags, mutations, variable declarations) is preserved in place. Runtime values flow through as named references.
